Program Arguments

The following java program receives as input a single argument:

public class TestArgument {

     public static void main (String[] args) {
         if (args.length != 1)
             System.out.println("Wrong number of arguments");
             else System.out.println("The value of the argument is: "+args[0]);
     }
}

To compile the program type
javac TestArgument.java

To run the program type
java TestArgument value

where value is any string. The program will output the value specified. However, if no argument is given or if two or more arugments are given the program will print an error message. If you use Eclipse to runt he above program, please read this Eclipse tutorial.
